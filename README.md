# Site institucional — Palhares Tech

Site comercial responsivo para apresentar serviços, projetos e soluções digitais da Palhares Tech.

## Executar localmente

Abra `index.html` diretamente ou inicie um servidor local:

```powershell
python -m http.server 4173
```

Depois acesse `http://localhost:4173`.

## Estrutura

- `index.html`: conteúdo e estrutura semântica.
- `styles.css`: identidade visual, responsividade e animações.
- `app.js`: menu, scroll progressivo, revelações e microinterações.
- `assets/`: versões vetoriais oficiais da marca Palhares Tech.
